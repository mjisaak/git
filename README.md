# git

## Clone a Github repository:
```bash
git clone https://github.com/mjisaak/git.git
```

## Retrieve git status:
```bash
git status
```

## Stage files for the next commit:
```bash
git add .
```

## Commit staged files:
```bash
git commit -m "added content"
```

## Push local commits to the remote repository:
```bash
git push
```
